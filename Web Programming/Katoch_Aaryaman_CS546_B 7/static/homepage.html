People Finder

This website searches for people based on id or name provided by the user.It has three pages he main page of this application will provide a search form to start a search on the dataset,this page will search through the people.json dataset and return up to 20 matching results (Sorted by ID), where either the firstName or lastName contains the provided request form param,this page will show all the details of the person with the id matching the provided URL param, id
